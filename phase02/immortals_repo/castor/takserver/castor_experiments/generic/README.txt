1) Copy query files to examples/ folder.

2) Set target relation name and files containing examples.

Open run.sh.
Set value of TARGET to target relation name.
Set value of POS_EXAMPLES to file containing positive examples.
Set value of NEG_EXAMPLES to file containing negative examples.

3) Run Castor: run.sh.

Output will be written to out/out.txt.
Learned definition will be written to out/definition.txt.
